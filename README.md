This is a simple JavaScript quiz game that runs completely inside the browser console.
It asks the user a series of questions using prompt(), checks the answers, gives instant feedback with alert(), and shows the final score at the end.

Features
	1.Runs fully in the browser console
	2.Uses prompt() for input and alert() for feedback
	3.Cleans user input with .trim() and .toLowerCase()
	4.Loops through an array of questions
	5.Tracks and displays the final score
